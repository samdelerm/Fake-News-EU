# Fake-News-EU

Site web statique sur la désinformation en Europe, prêt à être hébergé via GitHub Pages.

## Nouveautés

- 11 intox fréquentes sur l'UE, chacune confrontée à des faits vérifiables.
- Section de graphiques (canvas) avec chiffres clés pour contredire les rumeurs.
- Bibliographie renforcée vers des sources institutionnelles (UE, Eurostat, BCE, Cour des comptes, etc.).

## Contenu

- [index.html](index.html) : structure et contenu éditorial
- [styles.css](styles.css) : design moderne inspiré de l'Europe
- [script.js](script.js) : interactions (navigation, filtres, animations)

## Lancer en local

1. Ouvrir le dossier dans VS Code.
2. Démarrer une extension de serveur local (par exemple Live Server) ou ouvrir directement `index.html` dans le navigateur.

## Déploiement GitHub Pages

1. Pousser les fichiers sur la branche `main` du repo.
2. Aller dans **Settings > Pages** sur GitHub.
3. Dans **Build and deployment** :
	- **Source**: `Deploy from a branch`
	- **Branch**: `main` / `/ (root)`
4. Sauvegarder.
5. Attendre la publication, puis ouvrir l'URL fournie par GitHub Pages.

## Sources utilisées dans le site

Les références détaillées sont affichées dans la section « Sources & méthode » du site (Commission européenne, EDMO, EUvsDisinfo, etc.).
